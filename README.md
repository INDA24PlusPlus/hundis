## Utvecklingsmiljö
Dev-miljön har hot-reloading, så du behöver inte bygga om containrarna när du gör ändringar i koden.
### Gör så här!
1. Installera docker
2. Starta `docker compose -f compose.dev.yml up -d`
3. Avsluta `docker compose -f compose.dev.yml down`
